### 第 2 題：定義屬性與方法 (配分 40 分)

**【設計目的】**
練習如何在類別中定義物件的狀態（屬性）與行為（方法），並在主程式中把物件「實例化 (Instantiate)」出來。這是一道應用題，需要稍微思考。

* **作答檔案**：`Q2_Fish.java`
* **任務說明**：
    1. 在 `Q2_Fish` 類別中，宣告兩個屬性：
        * `name`（字串 `String` 型態，代表魚的名稱）
        * `weight`（浮點數 `double` 型態，代表重量）
    2. 撰寫一個名為 `displayInfo()` 的方法。當呼叫這個方法時，要在畫面印出：`這隻魚的名字是：[name]，重量是：[weight]公斤`。
    3. 在 `main` 方法中：
        * 產生一個 `Q2_Fish` 的物件（命名為 `myFish`）。
        * 將這隻魚的 `name` 設為 `"黑鮪魚"`，`weight` 設為 `250.5`。
        * 呼叫這隻魚的 `displayInfo()` 方法。
* **預期執行結果**：
    ```text
    這隻魚的名字是：黑鮪魚，重量是：250.5公斤
    ```

### 第 3 題：建構子 (Constructor) 的應用 (配分 20 分)

**【設計目的】**
這題稍難一點。學習使用「建構子」，讓你在產生物件的「當下」就能直接賦予它初始值，讓程式碼更簡潔。

* **作答檔案**：`Q3_Turtle.java`
* **任務說明**：
    1. 在 `Q3_Turtle.java` 中，已經幫你宣告好 `species` (品種) 與 `age` (年紀) 兩個屬性。
    2. 請撰寫一個**建構子 (Constructor)**，讓程式在建立海龜物件時，必須傳入 `species` 與 `age` 這兩個參數，並將傳入的值指定給類別的屬性。
    3. 在 `main` 方法中，利用你寫好的建構子，直接建立一隻品種為 `"綠蠵龜"`、年紀為 `50` 的海龜物件。
    4. 呼叫 `showDetails()` 方法印出資訊。
* **預期執行結果**：
    ```text
    品種：綠蠵龜，年紀：50歲
    ```
